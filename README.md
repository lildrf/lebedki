# lebedki
# Сайт производителя силовых лебедок

Сайт создан с использованием Bootstrap 5. Включает 5 основных страниц и страницу ошибки 404.

## Особенности проекта
- Полностью адаптивный дизайн
- SEO-оптимизированная структура
- Семантическая верстка
- Валидный HTML5/CSS3
- Кроссбраузерная совместимость

## Структура проекта
- index.html - Главная страница
- about.html - О компании
- services.html - Услуги
- portfolio.html - Портфолио
- contacts.html - Контакты
- 404.html - Страница ошибки
- css/style.css - Стили
- js/script.js - JavaScript
- robots.txt - Файл для поисковых систем
- sitemap.xml - Карта сайта

## Используемые технологии
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## SEO оптимизация
На всех страницах корректно прописаны:
- Meta description
- Meta keywords
- Title
- Alt теги для изображений
- Семантическая разметка
- robots.txt
- sitemap.xml

## Запуск проекта
Откройте файл index.html в браузере или разместите файлы на веб-сервере.

Все тексты написаны специально для сайта производителя силовых лебедок и ориентированы на целевую аудиторию - промышленные предприятия.
